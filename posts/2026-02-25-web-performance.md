# Web 性能优化指南

好的性能是用户体验的关键。

## 优化指标

### Core Web Vitals
- LCP (Largest Contentful Paint)
- FID (First Input Delay)
- CLS (Cumulative Layout Shift)

## 优化技巧

1. **图片优化** - 使用现代格式（WebP）
2. **代码拆分** - 按需加载 JavaScript
3. **缓存策略** - 合理配置浏览器缓存
4. **CDN 使用** - 加速静态资源分发
